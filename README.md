# Worth Work 

This is a simple script that was designed to be used with the Scriptable app on the iPhone that allows you to make informed purchasing decisions based on the cost of something compared to how much work that item is worth

## Features

- Stores wage information to use in calculations
- Computes how much work or how many hours it would take to afford the item

- Gives you option to provide temporary wage information to use in the calculations

## Installation
- The code should be available for download within the Scriptable app on iPhone. It should be under the same name and should be ready to use once downloaded

- The code in the worth_work javascript file can be copied and pasted in your own script within the Scriptable app on the iPhone
